Construcao do codigo para o formulario

Comessamos abrindo uma teg <form></form> que vai englobar todo o conteudo do formulario
Dentro do 'form' colocamos uma <fieldset></fieldset> para cada grupo de informações

Para campo texto usamos <input type="text" name="tnome do campo" id="cnome do campo" size="valor" maxlength="valor" placeholder="descrição">

    Componentes das linhas de comando
  • input
    • entrada de dados
  • type="---"
    • tipo de dados
      parametros para 'type="---"
    • text
      • entrada de textos como, 'nome,endereço...'
    • numero
      • dados numericos
    • email
      • confere se o email foi digitado de forma correta
    • password
      • digitação de senha, não mostra os caracteres digitados, so aparece asteriscos
    • range
      • cria uma barra graduada com 'min="x" para minimo; 'max="x" para maximo 'etep="x"'para graduação
    • checkbox
      • cria uma caixa de checagem para marcar itens selecionados
    • color
      • da acesso a caixa de cores para o cliente escolher a cor
    • submit
      • cria um botão cujo valor sera definido em 'script'
  • name="tnome do campo"
    • o 't' e pada diferenciar do 'id'que e preenchido com o mesmo nome
  • id="cnome do campo
    • o 'c' e para diferenciar do 'name'
  • size="valor"
    • tamanho maximo da caixa de texto, não limita os caracteres
  • maxlength="valor"
    • limita a quantidades de caracteres a ser digitado
  • radio
    • escolha de opção tipo sim, não ; nasculino , feminino...
  • date
    • cria um campo data que abre um calendario quando clicar no campo
Para que uma opção fique marcada por padrão colocamos o parametro 'checked' depois de 'id'
> Para campo 'email' <input type="email"name="temail" id="cemail"size="20" maxlength="40"placeholder="nome@xxx.com.br">
> para campo 'senha' <input type="password"name="tsenha" id="csenha"size="8"maxleng="8"placeholder="8 digitos">
> Para campo 'numero' <input type="number" name="tnum" id="cnum" min="0" max="99999">
    Complementos de 'type="number"
  • min="x"
    • valor minimo de entrada, ex. 'min="1" ' não aceita valor menor que 1
  • max="x"
    • valor maximo, ex. max="999" não aceita valor maior que '999'

<label for="cmasc">-masculino </label> <input type="radio" name="tsexo"id="cmasc">

. Campo 'mensagem'

Mensagem usa uma teg <textarea>
Cria um campo com varias linhas diferente do 'type text'
<textarea name="tmen" id="cmen" cols="30" rows="10"> </textarea>
tem os valores:
'cols'determina a altura padão
'rows' determina a largura padrão da caixa
